Wakeup

When I moved to Dallas, it didn't take me long to realize I needed to regularly check the traffic for my commute in the morning. Accidents, weather, and holidays all affected the amount of time it would take me to get to work on a given day. I started checking the route on the Apple Maps app each morning. This was a manual step, but I could check it on the way to the car, so not a huge deal. After a few weeks, I felt like the route it was giving me wasn't the most efficient. It had me take the highway which, during the morning rush hour, was stop-and-go all the way. Luckily, Google Maps has a feature that allows you to drag and drop custom waypoints on a route. So one night, I expiremented with this feature and found a route that I considered better. The only problem was that this was a custom route I created and was never one of the standard options. Well, that's what APIs are for!

Google offers a Directions API which allows custom route creation as described above. It also allowed me to avoid the toll road, which takes the same amount of time as the highway during rush hour, but you have to pay! After I got the directions working, I realized there's one other thing I check every morning: the weather. OpenWeatherMap provides several APIs that are sufficient. So, this turned into a pretty useful automation effort that I could use every morning.

One interesting (depending on you're definition of interesting) note deals with running this every day. I thought I would just use the Windows Task Scheduler to run this every morning. Since this would run after having my laptop idle overnight, I needed to allow the task to use a wake timer. To my surprise, this option did not appear in the power settings. After researching, I found that my laptop uses the Modern Standby power model, which means the computer only wakes when there is action required (OS maintenance/user wake), AKA no wake timers! On one forum, I saw someone suggest using an electircal timer to supply power to the laptop at a ceratin time and simulate a wake timer. While I love the ingenuity, I am no electrictian. At that point, I had some expereince with AWS and I was familiar with Lambda functions, so I went with that route. I set my function up with an EventBridge trigger to run every day at 7 AM, and that was that.
